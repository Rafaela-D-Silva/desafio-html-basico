<h1> até <h6>

Essas tags são usadas para títulos e subtítulos. O <h1> é o título principal da página — o mais importante. Já o <h6> é o de menor destaque. É como se fossem os níveis de títulos de um trabalho escolar: o título da capa é <h1>, os capítulos são <h2>, os subtópicos <h3>, e assim vai até <h6>.

Essas tags ajudam a organizar o conteúdo para o leitor e para os mecanismos de busca, como o Google. Usar corretamente melhora a leitura e o SEO da sua página.

<p>

A tag <p> é usada para criar parágrafos. É uma das mais básicas e importantes para escrever textos na web. Sempre que você quiser separar um bloco de texto, usa <p> no começo e </p> no final.

Ela organiza seu conteúdo em blocos legíveis e é essencial para deixar seu site bonito e fácil de entender.

<mark>

Essa tag serve para destacar um trecho do texto, como se você tivesse passado um marca-texto amarelo por cima. Pode usar quando quiser chamar atenção para uma palavra ou frase importante.

Visualmente, ela pinta o fundo do texto de amarelo.

🔙 Voltar ao índice

<small>

Essa tag diminui o tamanho da fonte do texto dentro dela. Pode ser usada para observações, rodapés, ou avisos pequenos que não precisam chamar atenção.

Ela não deixa o texto ilegível, só um pouco menor.

<i>

Usada para deixar o texto em itálico. Pode ser útil para palavras estrangeiras, nomes científicos, ou quando você quiser dar ênfase com estilo.

Visualmente, inclina o texto para a direita.

<u>

Essa tag sublinha o texto. Ela não é tão comum hoje em dia, porque o sublinhado pode confundir o usuário com links, mas ainda pode ser útil em alguns contextos.

<strong>

Deixa o texto em negrito e indica que aquele conteúdo é importante. É diferente do <b> (que só deixa em negrito), pois o <strong> também dá significado semântico de importância.

<ol>, <ul> e <li>

Essas tags são usadas para listas. <ol> é lista ordenada (numerada), <ul> é lista não ordenada (com bolinhas), e <li> é cada item da lista.

Elas são muito úteis para organizar informações de forma clara.
